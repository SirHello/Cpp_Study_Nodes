### using与构造函数
在C++11中，派生类能使用using重用其直接基类的构造函数。

派生类定义的构造函数会覆盖基类的重名构造函数。

对于基类的每个构造函数(默认，拷贝，移动构造函数除外)，编译器都生成一个与之对应（形参列表完全相同）的派生类构造函数。
```C++
class A
{
public:
    A() {}
    A(int a) {}
};

class B : public A
{
public:
    using A::A;
    //等价于
    //B() :A() {}
    //B(int a) :A(a) {}
};
```
